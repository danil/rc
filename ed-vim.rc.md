# Vim rc

https://vim.rtorr.com

## Undo/redo

* `u` Undo
* `U` Redo

## Delete/cut

* `x`  Delete character
* `dd` Delete a line

## Mode

* `i`          Insert
* `v` or `C v` Visual

## Search

`/`

## Unhighlight

`:noh`

## Yanks

* `Y` or `yy`
* `10Y`

## Paste

`P` or `p`

## Reload/reread file

`:e`

## Sort visual lines

`:sort`

## Move cursor

* `^` Line begin
* `$` Line end

## Scroll

* `C e` Down
* `C y` Up

## Buffer

* `:bd` Delete

## Session

* `:mks!`               Save
* `:source Session.vim` Load

## Autocomplete

`C n`
